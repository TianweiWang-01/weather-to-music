# weather to music
SkyMelody is an immersive weather-based writing studio. I built it as a browser-based creative workspace where users can choose a weather mood and write in an atmosphere that matches that mood. Instead of being just a normal text editor, each weather option changes the background, mood description, music suggestions, and AI assistant prompt, so the page feels more like a small themed writing room.
The application has a writing board with local autosave, live word and character counts, focus mode, and a countdown timer. The timer supports both preset times and custom minute/second input. When the countdown ends, the page gives the user a clear reminder with particle effects, a glowing timer ring, and a flashing border around the writing area. This helps bring the user’s attention back to the draft.
I also added weather-matched music recommendations by using the iTunes Search API. When the user changes the weather mood, the app searches for songs that fit that mood and displays the track name, artist, cover image, and playable preview if available. If the API does not respond, the app still shows local fallback recommendations, so the music section will not look empty.
For the AI writing companion, the frontend sends the current title, draft content, word count, and selected weather mood to a small Node.js backend. The backend is used to protect the OpenAI API key and then sends the request to the OpenAI API. After that, the writing suggestions are returned to the page. This design allows the frontend to be hosted on GitHub Pages while keeping the private API key on the server side.
Overall, SkyMelody is a complete themed writing tool. Users can choose a writing atmosphere, write their draft, use a timer, get visual feedback when the session ends, listen to music that matches the mood, and ask an AI assistant for writing suggestions based on their current draft.
